Non, si vous avez effectué une saisie et que vous vous identifiez, les données que vous aurez saisies (vos données "de session") ne seront pas automatiquement sauvegardées pour autant. Pour sauvegarder vos données de session, vous devez d'abord vous identifier (ou vous enregistrer si c'est la première fois), puis cliquer sur l'onglet "Sauvegarder" du menu de droite (cet onglet n'apparaît qu'après que vous vous soyez identifié).
