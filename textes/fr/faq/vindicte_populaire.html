Le courriel que vous avez (le cas échéant) utilisé pour vous identifier sur ce site est conservé sous forme cryptée dans notre base de données. Ceci permet de vous identifier si vous souhaitez revenir ultérieurement sur le site, et vous permet de retrouver les données que vous auriez sauvegardées. Mais il est impossible, à partir de la forme cryptée, de retrouver le courriel initial. Par conséquent, les données conservées dans notre base ne permet pas de vous identifier : elles sont, de notre point de vue, anonymes.

Donc quand bien même l'envie prendrait à certains de vous faire la leçon ou de vous vilainement châtier, ils ne pourraient pas vous identifier à partir des données que vous auriez enregistrées sur ce calculateur.
