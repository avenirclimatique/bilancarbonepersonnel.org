La réponse est oui, les données sauvegardées dans le calculateur pourront faire l'objet de traitements statistiques. Les résultats de ces statistiques pourront même être publiés s'ils présentent un intérêt, un caractère de nouveauté, si le nombre de données leur confère suffisamment de représentativité, etc.

Rappelons par ailleurs que les données que les utilisateurs peuvent sauvegarder dans le calculateur sont anonymes. En effet, le courriel fourni par les utilisateurs pour s'identifier est stocké sous forme cryptée dans la base de données.
