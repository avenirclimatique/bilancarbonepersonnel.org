Vos données de session sont les saisies que vous pouvez voir en parcourant les pages de questions du calculateur.

Vos données de sauvegarde sont des sauvegardes de vos données de session, qui sont réalisées à votre demande.

Vos données de session ne sont conservées en mémoire par le calculateur que pendant une durée limitée, plus exactement elles sont conservées en mémoire tant que vous utilisez le calculateur, puis pendant environ une heure après que vous avez cessé de l'utiliser (passé ce délai, vos données de session sont perdues).

Vos données de sauvegarde sont conservées en mémoire de façon pérenne par le calculateur. Vous pourrez les retrouver lors d'une visite ultérieure.

Si vous souhaitez que le calculateur conserve vos saisies en mémoire de façon pérenne, afin que vous puissiez les retrouver lors d'une visite ultérieure, vous devez les sauvegarder (sous la forme de données de sauvegarde). Pour cela, vous devez préalablement vous connecter (en cliquant sur l'onglet "Se connecter" - qui disparaît une fois que vous êtes connecté) puis cliquer sur l'onglet "Sauvegarder" (qui apparaît une fois que vous êtes connecté).

Attention : lorsque vous effectuez des modifications sur vos saisies, celles-ci ne sont pas automatiquement enregistrées sous la forme de données de sauvegarde. Tout comme lorsque vous modifiez un fichier sur un ordinateur, vous devez vous-même sauvegarder toute modification de votre saisie si vous souhaitez qu'elle soit enregistrée en données de sauvegarde. L'aspect de l'onglet "Sauvegarder" vous permet de savoir si la dernière version de vos données de session a été sauvegardée (si l'onglet n'est pas souligné, c'est que vos données de session coïncident avec vos données de sauvegarde du même nom).

Vous pouvez réaliser plusieurs jeux de données de sauvegarde. Ceci vous permet de faire des tests, des comparaisons, ou de conserver en mémoire du calculateur le Bilan Carbone Personnel d'autres personnes qui partagent votre foyer.
